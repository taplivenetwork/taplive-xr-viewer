# Contributing

Thank you for your interest in contributing to `taplive-xr`!

## Getting Started
1.  **Fork** the repo on GitHub.
2.  **Clone** the project to your local machine.
3.  **Branch** for your feature: `git checkout -b feature/AmazingFeature`.

## Development Workflow
- Ensure you have the correct Unity version (check `README.md`).
- Follow the script style guide (see `GEMINI_STRATEGY.md` for AI standards).
- Test your changes in the Unity Editor using the `SampleScene`.

## Pull Requests
1.  Push to your fork: `git push origin feature/AmazingFeature`.
2.  Open a Pull Request.
3.  Describe your changes clearly.
4.  Link any relevant Issues.
